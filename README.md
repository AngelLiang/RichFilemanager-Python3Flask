### Dependencies ###
* Python3
* Flask
* Pillow (PIL) http://pillow.readthedocs.io/en/latest/
* Virtualenv

### How do I get set up? ###

* cd flaskfilemanager/flaskfilemanager/
* source flaskfilemanagerenv/bin/activate
* export FLASK_APP=FlaskApp.py
* python3 -m flask run
